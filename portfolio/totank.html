ToTank

In the 2018-19 school year I took a computer science class in which I created To Tank using Unity and C#

To Tank is a two player game using a single keyboard (or two if you have a spare you can hook up) For Player 1 you use wasd to move and space to fire Player 2 uses the arrow keys for movement and enter (or return depending on the keybord) to fire.

Windows download

Windows instructions

extract zip file then open the extracted folder and click TankGame.exe choose quality and resolution and click play.
